Setup Guide

Polysauce Desktop App

Copy successful traders on Polymarket with automated bot execution

01

Get Your Polymarket API Credentials

Set up your builder account and generate API keys to connect the bot

Polymarket Settings Page
1

Access Settings Menu

Navigate to Polymarket and click your profile icon in the top right corner to open the dropdown menu.

2

Go to Builder Settings

Click on "Builders" from the menu to access your builder account settings.

3

Copy Your Wallet Address

Your wallet address appears at the top. This is your public identifier on Polymarket.

0x7e9ef499ccf01ab3cc26fdf2add89cd6cc048e63
4

Generate API Keys

Click "+ Create New" to generate a new API key set. You'll receive an apiKey, secret, and passphrase.

⚠️ Security Warning

Never share your private key or API credentials! These keys give full access to your Polymarket account. Store them securely and never commit them to git repositories or share them in chat.

02

Save Your API Credentials

Copy and securely store all three API components

API Details Modal
1

View API Details

After creating keys, click "View Details" to see your credentials. This is the only time you'll see them!

2

Copy All Three Keys

You'll need to copy three separate values:

apiKey: 019cb697-24...
secret: W1YVJbbPO1...
passphrase: 7ed4f30175...
3

Store Securely

Save these in a password manager or secure notes. You won't be able to view them again after closing this modal.

⚠️ Important

Polymarket may disable keys that are leaked or shared. If your keys are compromised, immediately generate new ones and delete the old set.

03

Choose a Wallet to Copy

Find successful traders on the leaderboard to follow

Builder Settings & Keys
1

Visit the Leaderboard

Go to polysauce.xyz/leaderboard to see top performing traders ranked by performance metrics.

2

Look for High Win Rates

Prioritize traders with 90%+ win rates and consistent performance over time.

3

Check Volume & PnL

Review their total volume traded and profit/loss to ensure they're actively trading profitably.

4

Copy the Wallet Address

Click on a trader to view their profile, then copy their wallet address to use in the bot.

0x7e9ef499ccf01ab3cc26fdf2add89cd6cc048e63

✓ Pro Tip

Start with traders who have at least 50+ trades and 30+ days of track record. This shows consistent performance rather than lucky streaks.

04

Configure the Desktop App

Enter your credentials and set up copy trading parameters

Portfolio & Activity Monitor
1

Enter Your API Credentials

In the app settings, paste your apiKey, secret, and passphrase from Step 2.

2

Add Target Wallet Address

Paste the wallet address you want to copy from the leaderboard (Step 3).

3

Set Risk Parameters

Configure trade size limits, maximum position size, and stop-loss percentages to manage your risk.

4

Start the Bot

Click "Start" to begin monitoring the target wallet. The bot will automatically copy their trades in real-time.
